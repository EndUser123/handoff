# Handoff Examples

This directory contains example code demonstrating how to use the handoff package.

## Examples

### basic_usage.py

Demonstrates basic usage of the handoff system for creating and managing conversation handoffs between sessions.

## Running the Examples

```bash
cd examples
python basic_usage.py
```

## Requirements

- Python 3.12+
- handoff package installed
